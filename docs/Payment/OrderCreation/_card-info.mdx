> **brand** <span style={{color:" #7d8793"}}>String</span> <span style={{color:"#f19938"}}>Required</span>
>

>
> `MASTERCARD` >`VISA` >`AE` >`DISCOVER` >`JC`
>
> <br />
>
> **cardId** <span style={{color:" #7d8793"}}>String</span>
>
> If the payer's card information was requested to be saved for future use., payby will return the card's ID.
>
> Example value: 31658300
>
> <br />
>
> **last4** <span style={{color:" #7d8793"}}>String</span> <span style={{color:"#f19938"}}>Required</span>
>
> Last 4 digits of card number.
>
> Example value: 6345
>
> <br />
>
> **cardType** <span style={{color:" #7d8793"}}>String</span> <span style={{color:"#f19938"}}>Required</span>
>
> The possible values are:
>
> `DC`. Debit Card
>
> `CC`. Credit Card
>
> <br />
>
> **expMonth** <span style={{color:" #7d8793"}}>String</span> <span style={{color:"#f19938"}}>Required</span>
>
> Two -digit number, representing the card expiry month.
>
> Example value: 01
>
> <br />
>
> **expYear**<span style={{color:" #7d8793"}}>String</span> <span style={{color:"#f19938"}}>Required</span>
>
> Two -digit number, representing last two digits of the card expiry year.
>
> Example value: 22
